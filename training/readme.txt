2) Dado um array de ints, desenvolva um método que retorne o respectivo array ordenado por ordem crescente ou decrescente. O método que realiza a ordenação deverá apresentar a seguinte assinatura:

sortArray(int [] intArr, OrderEnum order)

Notas: 
* O argumento OrderEnum deverá ser do tipo Enum
* O método deverá fazer parte de uma class de utilitários (Utils)
* Deverá ter-se em conta a possibilidade o caller do método sortArray não providenciar uma ordenação. Por omissão considere ordenação decrescente
